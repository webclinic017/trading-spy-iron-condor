BATS Framework — Budget Enforcement Logic
✓ Daily budget: $0.83/day ($25/month ÷ 30)
✓ can_afford_model() estimates cost before each call (60/40 input/output split)
✓ Over budget → automatic downgrade chain: Opus → Kimi → Mistral → DeepSeek
⚠ CRITICAL tasks NEVER downgraded — trade execution always uses Opus
✓ Daily reset at midnight; monthly reset on 1st
✓ All selections logged → selection_log[] (last 1000 entries, audit-ready)
